Инструкция для запуска программы.

1) Открыть командную строку в папке проекта:
...\testForScaleApps\out\production\classes>

2) Запустить класс Main c аргументами: Пример java Main -a -i out_file.txt input_file.txt


--Порядок аргументов:
1-м передается аргумент ввода входной файл или '-', обязательный для ввода:
    агрумент '-' - это ручной ввод арифметического выражения с клавиатуры.

2-м передается выходной файл или '-', варианты (-i или -s), обязательный для ввода;
    агрумент '-' - это вывод ответа (чему равно введеное арифметическое выражение) в консоль.

--Арифметические операции:
    add - сумма 2х и более чисел;
    mul - произведение 2х и более чисел;
    mulPlus - произведение первых 2х чисел, суммируется с 3м числом.

--Оперции в консоли:
    help - выводит список арифметических операций;
    exit - выход из консоли.
